
# 📊 Student Performance Analysis (EDA & SQL + Dashboard)

## 📌 Overview

This project analyzes student performance data using **Exploratory Data Analysis (EDA)**, **SQL**, and **Business Intelligence techniques** to identify key factors affecting exam scores.

---

## 🎯 Objectives

* Clean and preprocess the dataset
* Perform exploratory data analysis
* Solve business questions using SQL
* Generate meaningful insights
* Build a dashboard for visualization

---

## 🗂️ Dataset

The dataset contains information about students such as:

* Student_ID
* Gender
* Hours_Studied
* Attendance
* Sleep_Hours
* Parental_Education_Level
* Internet_Access
* School_Type
* Exam_Score

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (MySQL / SQLite)
* Power BI / PowerPoint
* Jupyter Notebook
* GitHub

---

## 🔍 Exploratory Data Analysis (EDA)

### Data Cleaning

* Removed duplicates
* Handled missing values
* Converted data types

### Analysis Performed

* Distribution of exam scores
* Study hours vs performance
* Attendance impact
* Correlation analysis

---

## ❓ Business Questions (SQL)

* Does study time affect exam score?
* Does attendance impact performance?
* Do sleep hours influence marks?
* Does parental education matter?
* Does internet access improve performance?

---

## 🧠 Key Insights

* 📈 Higher study hours lead to better scores
* 📊 Attendance strongly impacts performance
* 💤 Balanced sleep improves results
* 🌐 Internet access slightly improves performance

---

## 📊 Dashboard

The dashboard includes:

* KPI metrics (Average Score, Total Students, Max Score)
* Study vs Performance trends
* Category-wise analysis
* Pass vs Fail distribution
